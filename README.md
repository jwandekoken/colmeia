# colmeia

Monorepo for the Colmeia platform (API + Web).

**Docs**

- `docs/colmeia.md` — Colmeia overview
- `docs/stack.md` — Tech stack overview
- `docs/api-architecture.md` — API architecture and module layout

**Local Development**

1. Install dependencies: `pnpm install`
2. Start API: `pnpm dev:api`
3. Start Web: `pnpm dev:web`
4. Start all: `pnpm dev:all`
